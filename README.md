# Java Age Program

This is a simple Java program to calculate age.

## Features
- Takes input from user
- Calculates age
- Displays result

## How to Run
1. Compile the program:
   javac age.java
2. Run the program:
   java age

## 🔢 Even Number Program

This program checks whether a number is even or odd.

### ▶️ How to Run
1. Compile:
   javac EvenCheck.java
2. Run:
   java EvenCheck

Enter a number: 4

## 🔢 Greatest of Three Numbers

This Java program finds the greatest among three numbers entered by the user.

### 📌 Features
- Takes three numbers as input
- Compares values using if-else
- Displays the greatest number

### ▶️ How to Run
1. Compile:
   javac GreatestOfThree.java
2. Run:
   java GreatestOfThree

### 💡 Example Output

Enter first number: 10  
Enter second number: 25  
Enter third number: 15  
Greatest number is: 25
